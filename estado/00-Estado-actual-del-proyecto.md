# Estado actual del proyecto

Fecha: 2026-01

Este documento describe el estado real del proyecto en el momento de su
documentación, con el objetivo de proporcionar contexto técnico suficiente
para entender decisiones, pausas y limitaciones si se retoma en el futuro.

## Situación general

En el estado actual no existe una estación RF fija ni una instalación
permanente operativa.

El proyecto se encuentra en una fase de baja actividad intencionada,
sin objetivos operativos a corto plazo ni presión por mantener continuidad.

## Equipamiento disponible

El equipamiento actual se limita a:

- Un equipo portátil (AnyTone), utilizado de forma esporádica
- Un receptor RTL-SDR, empleado para escucha y pruebas puntuales

No hay antenas instaladas de forma permanente ni sistemas radiantes dedicados.
Tampoco existe infraestructura asociada a una estación fija.

## Uso real del equipamiento

El uso ha sido principalmente exploratorio y de aprendizaje:

- Familiarización con equipos y software
- Pruebas básicas sin continuidad operativa
- Evaluación de interés y viabilidad personal del proyecto

No se ha consolidado una rutina de uso ni un entorno estable.

## Elementos no implementados

De forma consciente, no se han desarrollado los siguientes aspectos:

- Instalación de antenas
- Estación fija
- Operación regular
- Desarrollo de configuraciones avanzadas

Las razones principales son falta de prioridad, interés variable y ausencia
de un objetivo técnico claro que justifique la inversión de tiempo.

## Decisiones tomadas

- No forzar la actividad del proyecto
- Documentar el estado y las decisiones en lugar de acumular material
- Mantener el proyecto en pausa abierta, sin cerrarlo ni escalarlo artificialmente

La pausa se considera una decisión técnica y organizativa, no un abandono.

## Condiciones para una posible reactivación

El proyecto podría retomarse si se cumplen alguna de las siguientes condiciones:

- Aparición de un objetivo técnico concreto y acotado
- Interés renovado por un modo, experimento o tecnología específica
- Disponibilidad de una instalación mínima viable sin fricción significativa

Hasta entonces, el proyecto permanece documentado y estable.
