# rf-notes

Repositorio de documentación técnica y notas operativas relacionadas con RF,
radio y experimentación práctica, mantenido como cuaderno de trabajo y referencia
a medio y largo plazo.

El objetivo no es recopilar teoría ni material académico, sino documentar
configuraciones, pruebas, decisiones técnicas y evolución real del proyecto.

## Alcance del repositorio

Este repositorio puede incluir, según exista material relevante:

- Apuntes técnicos en Markdown derivados de uso real
- Configuraciones comentadas (software, CPS, herramientas, entornos)
- Resúmenes operativos y esquemas mentales orientados a uso práctico
- Pruebas y experimentos con resultados documentados
- Contexto y estado del proyecto para facilitar su reanudación tras periodos de inactividad

El contenido se añade únicamente cuando aporta valor técnico o contextual.

## Estructura

- `estado/`  
  Documentación de contexto: estado del proyecto, equipamiento disponible,
  decisiones técnicas tomadas y justificación de las mismas.

- `notas/`  
  Notas técnicas y evolución del proyecto: pruebas realizadas, configuraciones,
  conclusiones parciales y cambios de enfoque.

## Criterios de documentación

- Se documenta lo probado, no lo teórico
- Se prioriza la reproducibilidad y el contexto
- Se incluyen errores y resultados no satisfactorios si aportan información
- No se persigue completitud ni regularidad temporal

Este repositorio actúa como referencia técnica personal y registro de trabajo,
no como guía general.
